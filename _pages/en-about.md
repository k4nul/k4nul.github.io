K4NUL is a technical blog for security analysis, Rust learning notes, AI engineering operations, and DevOps workflows.

The site favors evidence-oriented notes: what was checked, what environment was used, what can be reproduced, and what remains a limitation.

## Main Topics

- Security: document malware analysis and practical security engineering.
- AI Engineering: Codex, Claude Code, harness engineering, token management, and agent workflows.
- Rust: setup, language fundamentals, debugging, CLI practice, and project structure.
- DevOps: Docker, Git, Jenkins, Kubernetes, and operational automation.
